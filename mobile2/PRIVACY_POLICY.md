# Política de Privacidade — Reflexo App

**Última atualização:** 2026-05-25
**Desenvolvedor:** dulkz

## Dados coletados
- Email (apenas para autenticação, opcional)
- Tempos de reação e resultados de partidas (para ranking global, apenas se logado)
- Username escolhido pelo usuário

## Dados NÃO coletados
- Localização
- Contatos
- Câmera ou microfone
- Dados de terceiros

## Uso dos dados
- Autenticação via Supabase (supabase.com)
- Exibição no ranking global do app
- Os dados não são vendidos ou compartilhados com terceiros

## Armazenamento
- Dados locais: AsyncStorage no dispositivo do usuário
- Dados online: Supabase (servidores na região padrão)
